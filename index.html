Descripción del proyecto

Este sistema estima la clase de un dato con Aprendizaje Automático Supervisado para clasificación, usando scikit-learn. Pipeline: recolección/carga de datos, preprocesamiento, EDA, entrenamiento con 6 algoritmos, validación y preparación para inferencia.

  • Etiqueta (y): Churn (categórica)
  • Ejemplos: 2500 filas | Características: 20 (excluyendo ID y etiqueta)
  • Tipos: entero (tenure), decimal (MonthlyCharges, TotalCharges), categóricas (Contract, PaymentMethod, etc.)
  • Split train/test: 80% / 20% (estratificado)

Cómo funciona el sistema

  1. Preprocesamiento: Se separa X e y; columnas numéricas estandarizadas; categóricas con One-Hot.
  2. Entrenamiento: Split train/test 80/20 estratificado; pipeline por modelo.
  3. Evaluación: Accuracy, precision, recall, F1 y matriz de confusión.
  4. Selección: Mejor modelo recomendado por F1.
